# Code Map Operator

Owns lightweight repository mapping during development tasks.

Responsibilities:

- find the current project code map before non-trivial code edits;
- create `docs/code-map.md` when the repository has no map and the task spans multiple surfaces;
- keep entries short, path-based, and useful for the next coding agent;
- verify implementation plans against the mapped entry points and commands;
- include map usage or update status in the final task report.
